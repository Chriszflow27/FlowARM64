\## ✨ Tagline

Automatiza tu suerte en KeyDrop — optimizado para ARM64.



\---



\## 📖 Descripción

\*\*FlowARM64\*\* es un bot diseñado para dispositivos \*\*Tanix W2 ARM64\*\* que se integra con \*\*Playwright\*\* para saltar la protección de \*\*Cloudflare\*\* y unirse automáticamente a los sorteos oficiales de \*\*KeyDrop\*\* por ti.



\---



\## 🚀 Características

\- 🔒 Bypass automático de Cloudflare con Playwright.  

\- 🎯 Detección de los 4 sorteos oficiales de KeyDrop.  

\- 🤖 Unión automática vía API PUT.  

\- 📡 Notificaciones en Discord para cada sorteo unido.  

\- ⚡ Optimizado para \*\*Debian 13 ARM64\*\*.
